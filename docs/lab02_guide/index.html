1. ศูนย์อาหารคอมเพล็กซ์

แหล่งรวมอาหารยอดนิยมของนักศึกษา ราคาย่อมเยา บรรยากาศคึกคัก อยู่ใกล้คณะวิทยาศาสตร์ (ปัจจุบันกำลังจะเข้าสู่การปรับปุรง)

2. คุ้มสีฐาน

พื้นที่อาคารชุดสำหรับจัดกิจกรรมหลากหลายรูปแบบ เป็นศูนย์ในการทำกิจกรรมต่างๆ ทั้งในส่วนของสถานที่และภาพลักษณ์ในการทำกิจกรรมในอนาคต รวมทั้งเป็นบริเวณของศูนย์กลางด้านวัฒนธรรม เป็นบริเวณที่จัดการเรียนรู้ที่หลากหลาย

3. ศูนย์ประชุมอเนกประสงค์กาญจนาภิเษก

อาคารขนาดใหญ่ทรงกูบเกวียน ตั้งอยู่ทางทิศตะวันออกจากบึงสีฐาน เหมาะสำหรับการจัดประชุมสัมมนา การแสดงนิทรรศการ การจัดกิจกรรมพิเศษต่าง ๆ

4. สถาบันขงจื้อ

สถาบันสอนภาษาและวัฒนธรรมจีนแก่ นักเรียน นักศึกษาและบุคลากรของมหาลัยขอนแก่น

แบบฟอร์มแนะนำสถานที่เพิ่มเติม
